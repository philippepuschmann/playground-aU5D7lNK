# Welcome!

This C# template lets you get started quickly with a simple one-page playground.

```C# runnable
// { autofold
using System;

class Hello 
{
   static void Increment(int i)
{
    i=i+1;
}

static void Main()
   {
int x = 1;
Increment(x);
Console.WriteLine("The value of x is " + x);
Console.Read();
    }
    }
```
